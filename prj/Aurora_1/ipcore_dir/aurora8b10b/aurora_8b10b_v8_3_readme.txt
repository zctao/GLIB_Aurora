3. NEW FEATURE HISTORY

  3.1 ISE

    v8.3  

    - Artix-7 family support 

    - Zynq family support 

    - CRC feature addition 

    - Hot-plug support for 7-series

    - ISE 14.3 software support

    v8.1  

    - Virtex-7/Kintex-7 GES silicon support 

    - Scrambler/Descrambler feature addition 

    - ISE 14.1 software support  

  3.2 Vivado

    v8.3 Rev 1 

    - Artix-7 IES silicon support 

    - Vivado 2012.4 software support

    - Autoupgrade feature

    v8.3  

    - Artix-7 family support 

    - Vivado 2012.3 software support

    v8.2  

    - Virtex-7 HT device support 

    - CRC feature addition 

    - Hot-plug support for 7-series

    - XSIM simulator support

    - Vivado 2012.2 software support (initial release)
